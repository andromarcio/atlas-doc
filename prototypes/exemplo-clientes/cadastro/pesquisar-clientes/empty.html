📋 Notas — Empty state (resultado de pesquisa vazio):
  • Tipo: "ausência de dados por ação da pessoa usuária (filtro)" — guia UX Writing p.214
  • Título positivo: foca no que pode ser feito, não no que faltou ("Nenhum cliente encontrado com esses filtros", não "Sua pesquisa não retornou resultados")
  • Lista de dicas: nunca deixar a pessoa num "beco sem saída" — guia p.215
  • Filtros permanecem visíveis para facilitar o ajuste sem recarregar a página
  • Se for primeiro uso (sem nenhum cliente cadastrado ainda), exibir empty-state diferente com botão "Cadastrar Cliente"
